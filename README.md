# 🍰 Blisskery — Bakery Ordering System

> A Java Swing desktop application for a bakery, featuring customer ordering, an admin dashboard, a cooking challenge mini-game, and MySQL-backed persistence. Built as an OOP course project.

---

## ✨ Features

### 👤 Authentication
- Login and registration system with role-based access
- Two roles: **Customer** and **Admin**
- Credentials stored in a MySQL database

### 🛒 Customer Panel
- Browse the full menu organized by category (Breads, Cakes, Cupcakes, Cookies, Pastries, Pies & Tarts, Brownies & Bars)
- Add items to cart and view running total
- Proceed to checkout via a **Cooking Challenge** mini-game
- Earn a **10% discount** by solving the challenge correctly
- Receipt generated after every order

### 🔐 Admin Dashboard
- Add new menu items with name, price, and category
- Delete existing menu items by name
- Status log panel showing all recent actions

### 🎮 Cooking Challenge Mini-Game
- A timed math puzzle (10 seconds) appears at checkout
- Solve `A + B × C = ?` correctly to unlock a 10% discount
- Tests order of operations (multiplication before addition)

### 🗄️ Database
- MySQL backend via JDBC
- Auto-creates schema and seeds default data on first run
- 27 default menu items across 7 categories seeded automatically

## ⚙️ Setup & Running

### Prerequisites
- Java JDK 8 or higher
- MySQL Server running locally on port `3306`
- [MySQL Connector/J](https://dev.mysql.com/downloads/connector/j/) JAR downloaded

### 1. Database Setup
Create the database in MySQL (the app creates the tables automatically):
```sql
CREATE DATABASE foodie_palace;
```
Default credentials used by the app:
- Host: `localhost:3306`
- Database: `foodie_palace`
- User: `root`
- Password: `root`

> To change these, edit the constants at the top of `FileStore.java`.

### 2. Update the JAR Path
Open `scripts/build.ps1` and `scripts/run.ps1` and update `$JarPath` to point to your local MySQL Connector/J `.jar` file:
```powershell
$JarPath = "C:\path\to\mysql-connector-j-x.x.x.jar"
```

### 3. Build
```powershell
.\scripts\build.ps1
```
This compiles all `.java` files from `src/` into `bin/`.

### 4. Run
```powershell
.\scripts\run.ps1
```

### Default Admin Login
| Username | Password | Role |
|----------|----------|------|
| `admin` | `admin` | ADMIN |
